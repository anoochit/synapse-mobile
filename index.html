Welcome to Synapse Lite!

Synapse Lite! is a digital signage application for Android, you can use an Android Box which attach to TV to make as a digital signage. the application can use 2 types of data source 1. Image - use for image slide show change an image every 10 secs 2. Web Site - use your own content from any web site just provide URL

How to use

The application bundle example image contents and select data source as image. To change you own content just copy your image files to /sdcard/Synapse.

If you want to use web content, go to setting page by "click at clock" after that click at Web site URL enter your own URL and change data source to Web site. You can use dashboard services like geckoboard, duckboard etc to provide the content.

The application also support text marquee, and digital clock. You can change text color, background color or set color to none for transparent.
